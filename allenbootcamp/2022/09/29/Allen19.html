Foundation

  • Swift 언어만 가지고는 할 수 있는 것이 거의 없다.
  • 기초적인 기능들은 Foundation 라이브러리를 import해야 한다.

어떤 기능들이 있을까요?

  • Obj-C 관련 기능 / 타입 / 클래스 / 컬렉션 …
  • 멀티 쓰레드 / 네트워크 / JSON / Selector / 런루프 / NSData 등

iOS의 프레임워크 계층

  • 4개의 계층으로 구성
  1. Cocoa Touch Framework
    • UIKit / GameKit / MapKit / NotificationCenter / …
    • 앱의 겉모습에 관련된 기능 (사용자와 관련)
  2. Media Framework
    • CoreGraphics / AVFoundation
    • 미디어 기능 구현 필요 기능
  3. Core Services Framework
    • Foundation
    • 시스템 서비스 제공
  4. Core OS Framework
    • CoreBluetooth / LocalAuthentication / Security
    • 보안 / 하드웨어 연동
