연산자

연산자(Operator) 기초

  • a + b
    • a, b: 피연산자(operand)
    • +: 연산자(operator)
  • 단항연산자: 단일 대상의 바로 앞/뒤에 위치
    • +a, -b, !a
  • 이항연산자: 두 개의 대상 가운데 위치
    • a + b, a >= b
  • 삼항연산자: 세 개의 대상 가운데 위치
    • a ? b : c
  • 연산자 사이의 공백 규칙
    • 단항연산자는 공백 없이 붙여서 사용
    • 이항/삼항 연산자는 일반적으로 양쪽에 공백 1칸씩 추가하여 사용

기본연산자

할당 연산자 (Assignment Operator)

  • 오른쪽의 값을 왼쪽에 대입
let num2 = 10
var num1 = 5

산술 연산자 (Arithmetic Operator)

  • 기본적으로 Swift에 내장되어 있는 연산
    • +: 더하기
    • -: 빼기
    • *: 곱하기
    • /: 나누기
    • %: 모듈러 (나머지)
var a = 456
var b = 123

a + b
a - b
a * b
b / a
b % a
모듈러(%)의 사용
var watchingTime: Int = 70

var hour = watchingTime / 60
var minute = watchingTime % 60
print("당신은 \(watchingTime)분. 즉, 총 \(hour)시간 \(minute)분을 시청하셨습니다.")

var num = 7
var array = ["1", "2", "3"]
array[0]
array[num % 3]
나누기의 사용은 항상 주의
a = 4
b = 5

var newResult: Double
newResult = Double(a / b) // 0.0
newResult = Double(a) / Double(b) //0.8

복합 할당 연산자 (Compound Assignment Operators)

value += 10 // value = value + 10
value -= 5 // value = value - 5

비교 연산자 (Comparison Operator)

  • 결과적으로 문장의 평가값은 참/거짓으로 도출
a = 456
b = 123

a == b // False
a != b // True
a > b
a >= b

논리 연산자 (Logical Operator)

  • !: Logical NOT Operator
  • &&: Logical AND Operator
  • ||: Logical OR Operator

접근 연산자

  • .: 하위 개념에 접근
Int.random(in: 1...3)
var number = Int.random(in: 1...5)

var name = "Steve"
print(name.count) // 5

연산의 우선순위

  1. *, /, % (곱하기 / 나누기)
  2. +, - (더하기 / 빼기)
  3. <=, >, ==, != (비교)
  4. &&, || (논리)
  5. 삼항연산자
  6. =, += (할당, 복합 할당)
