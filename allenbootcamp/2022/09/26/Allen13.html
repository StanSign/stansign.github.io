inout 파라미터

  • 함수 내의 파라미터는 기본적으로 값 타입이고, 임시 상수이기 때문에 변경이 불가하다.
var num1 = 123
var num2 = 456

func swap(a: Int, b: Int) {
	var temp = a
	a = b
	b = temp
}
swap(a: num1, b: num2)

그렇다면 참조 타입으로는 값을 전달하지 못할까요? 그럴리가요.. 여기서 inout 파라미터가 등장합니다.

var num1 = 123
var num2 = 456

func swapNumbers(a: inout Int, b: inout Int) {
	var temp = a
	a = b
	b = temp
}
swapNumbers(a: &num1, b: &num2) // &: 주소값
print(num1, num2) // 456, 123

주의점

  • 상수나 리터럴을 전달하는 것은 불가능
  • 파라미터의 기본값 선언을 허용하지 않음
  • 가변 파라미터로 선언하는 것 또한 불가능
