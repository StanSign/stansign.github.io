삼항연산자와 범위연산자

삼항연산자 (Ternary Conditional Operator)

  • if문보다 제한적인 조건문
  • 하지만 용도는 확실!
  • [조건] ? [참일때의 결과] : [거짓일때의 결과]
var a = 10

if a > 0 {
	print("Positive")
} else {
	print("Negative")
}
// 위 예시는 아래와 같다.
a > 0 ? print("Positive") : print("Negative") // Positive

let result = score >= 70 ? "Pass" : "Fail"
  • 조건에 따라 선택이 두 가지일 경우, 삼항연산자가 유용하다!

범위연산자 / 패턴매칭연산자

범위연산자

  • 제한된 숫자의 범위를 표기하기 위한 연산자
    • 자체가 특별한 타입
  • 1...10: 1 ~ 10

규칙

  • 내림차순 형식 표기 불가능
  • 실수 형식 범위 가능

폐쇄범위연산자 (Closed Ranged Operator)

  • 1...10: 1 이상 10 이하, ClosedRange<Int>.Type
  • 1...: 1 이상, PartialRangeFrom<Int>.Type
  • ...10: 10 이하, PartialRangeThrough<Int>.Type

반폐쇄범위연산자 (Half-Open Range Operator)

  • 1..<10: 1 이상 10 미만, Range<Int>.Type
  • ..<10: 10 미만, PartialRangeUpTo<Int>.Type

활용

  1. for문과 함께 사용
for index in 1...5 {
	print("\(index) times 5 is \(index * 5)")
}

let names = ["Anna", "Alex", "Brute", "Carner"]
let count = names.count
for i in 0..<count {
	print("Person \(i + 1) is called \(names[i])")
}
  1. 배열의 서브스크립트 문법과 함께 사용
names[...2]
names[..<1]
  1. switch문의 케이스에서 사용
    • 패턴 매칭
var num = 23
switch num {
case 10...19:
	print("10대")
case 20...29:
	print("20대")
case 30...39:
	print("30대")
default:
	print("그 외")
}

패턴매칭연산자 (~=)

  • 숫자가 범위 내에 있는 지 확인하는 연산자
    • 범위연산자와 함께 쓰인다.
  • a...b ~= age
  • 오른쪽에 있는 표현식이 왼쪽의 범위에 포함되면 true, 아니라면 false 
var n = 20

if n >= 10 && n <= 100 {
	print("10 이상, 100 이하입니다.")
}
// 위 예시는 아래와 같다!
if 10...100 ~= n {
	print("10 이상, 100 이하입니다.")
}
