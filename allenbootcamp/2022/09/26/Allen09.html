반복문

for문

  • 특정 조건을 만족하는 동안 반복
  • for [item] in [범위 / 컬렉션] { [반복될 코드] }
for index in 1...5 {
	print("\(index)번째 반복 중")
}

주의!

  1. for문에서 선언하는 변수는 상수이다!
    • 반복 상수 (임시 상수)
// 변수로 사용하고 싶다면?
for a in 1...10 {
	var b = a // 따로 선언해서 저장해야함
}

var name = "홍길동"
for name in 1...5 {
	print(name, terminator: " ")
}
// 1 2 3 4 5
for n in 1...5 {
	print(name, terminator: " ")
}
// 홍길동 홍길동 홍길동 홍길동 홍길동
  1. for문에서 선언한 상수는 외부에서 사용할 수 없다!
for a in 1...10 {
	print(a)
}

print(a) // Error : Cannot find 'a' in scope

와일드 카드 패턴

  • 언더바의 등장
  • _: 생략을 의미
for _ in 0...10 {
	print("Hello!")
}
// 위 예시는 반복을 사용하지만 변수를 사용하지 않는다.
// 따라서 굳이 변수를 생성해 줄 필요가 없으니 _(언더바)를 사용!

범위연산자를 거꾸로 쓰고 싶을 때

  • reversed
let a = (1...10).reversed()
print(a) // 10 9 8 ... 2 1

반복문은 컬렉션 타입에서도 사용할 수 있다!

  • 컬렉션에서 각 아이템을 차례대로 뽑아내서 사용할 때 활용 가능
let list = ["Swift", "Programming", "Language"]

for str in list {
	print(str)
} 
// "Swift"
// "Programming"
// "Language"

for chr in "Hello" {
	print(chr, terminator: " ")
	// terminator: \n 대신 사용
}
// H e l l o

반복문의 조건범위에 사용 가능한 특수 함수들

reversed

  • 역순으로
for number in (1...5).reversed() {
	print(number, terminator: " ")
}
// 5 4 3 2 1

stride

  • 건너뛰며 뽑아내기
  • stride(from: 1, to: 15, by: 2) == [1, 3, 5, 7, 9, 11, 13]
    • to 에 해당되는 값은 해당되지 않는다.
  • StrideTo<Int>.Type, StrideThrough<Int>.Type 
for number in stride(from: 1, to: 15, by: 2) {
	print(number, terminator: " ")
}
// 1 3 5 7 9 11 13

while문

  • 마찬가지로 조건을 만족하는 동안 반복
  • while [조건] { [코드] }
  • for문과 다른 점은 조건을 만족하는 동안 무한히 반복한다.
    • 따라서 반복되는 코드 안에서 조건의 변화가 이루어져야 한다!
var num = 0
while num <= 50 {
	print(num)
} // 무한 루프..!

while num <= 50 {
	print(num)
	num += 1
} // 0..<50 반복

repeat-while문

  • repeat { [코드] } while [조건]
  • while문과의 차이점
    • 일단은 한 번 실행하고 나서 조건을 판단
    • 조건 판별의 시점 (while: 반복코드 전에, repeat-while: 반복코드 후에)
  • 다른 언어에서는 do-while문이라고 불린다.
var i = 1
repeat {
	print("\(3) * \(i) = \(3 * i)")
	i += 1
} while i <= 9

제어전송문 (Control Transfer Statement)

continue

  • 이번 반복은 넘어가고, 다음 반복문으로 진행시킨다.
for num in 1...20 {
	if num % 2 == 0 {
		continue
	}
	print(num, terminator: " ")
}
// 1 3 5 ... 19

break

  • 반복문을 중단하고 탈출한다.
for num in 1...20 {
	if num % 2 == 0 {
		break
	}
	print(num, terminator: " ")
}
// 1

주의

  • 중첩 사용 시의 적용 범위
    • 인접한 반복문에만 적용된다!
for i in 0...3 {
    print("OUTER \(i)")
    for j in 0...3 {
        if i > 1 {
            print("  j :", j)
            continue
            //break
        }
        print("  INNER \(j)")
    }
}

Labeled Statements

  • Label이 매겨진 문장과 continue, break의 사용
  • 반복문에 Label을 매겨 제어전송문을 제어
OUTER: for i in 0...3 {
    print("OUTER \(i)")
    INNER: for j in 0...3 {
        if i > 1 {
            print("  j :", j)
            continue OUTER
            //break OUTER
        }
        print("  INNER \(j)")
    }
}
