프로그래밍 기본

등호(=)의 의미

  • 프로그래밍에서 사용하는 등호(=)는 할당의 의미이다.
    • 할당연산자 
      var a = 5
a = a + 7
  • 등호 기준으로 오른쪽에 있는 무언가를 왼쪽에 담는다
  • 같다에는 == 사용
    • == : 같다
    • != : 같지 않다

대문자와 소문자의 엄격한 구분

  • 대부분의 첫글자는 소문자로 시작한다. 
    var nameOfYou = "홍길동"
  • 카멜케이스(camel-case)
    • 언어에 따라 다르다

키워드(keyword)

  • 약속어 (Swift에서 특별한 의미로 사용하겠다고 미리 정해놓은 단어)
  • 약 70여개 존재
    • 참고: [[https://www.tutorialkart.com/swift-tutorial/swift-keywords/]]
  • 다른 의미로 사용 불가능 
    var
func
if

주석 (comment)

  • 컴퓨터가 코드를 읽지 않아도 된다고 알려주는 부분
  • //: 한 줄 주석
  • /* */: 여러줄 주석 
    print("Hello, Swift!")
// print("Hello, Swift!")
/* 
print("Hello, Swift!")
/* 
Nested Comment
*/
print("Hello, World!")
*/
