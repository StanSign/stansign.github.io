앨런 Swift문법 마스터 스쿨 (온라인 BootCamp - 2개월과정)

키워드

  • Swift에서 특별한 의미로 사용하겠다고 미리 정해놓은 단어
  • ex) var, let, if, true

리터럴 (Literals)

  • 코드에서 고정된 값으로 표현되는 데이터 그 자체
var a = 4 // <정수 리터럴> 값인 4를 넣었다.
var b: Int = 8

var name = "홍길동" // <문자열 리터럴> "홍길동"
var double = 3.14 // <double 리터럴> 3.14

식별자 (Identifier)

  • 변수, 상수, 함수, 사용자 정의 타입의 이름
var name1 = "전지현" // name1
var name2 = "임꺽정" // name2

토큰 (Token)

  • 코드의 가장 작은 단위 (식별자, 키워드, 구두점, 연산자, 리터럴)
  • ex) var, !=, 123

표현식 (Expression)

  • 값, 변수, 연산자의 조합으로 하나의 결과값으로 평가되는 코드 단위
var n = 5 // 문장
17 // 표현식
n  // 표현식
n + 7 // 표현식
n < 5 // 표현식

문장 (Statement)

  • 특정 작업을 실행하는 코드 단위
var n2 = 3 // 문장
print(n)   // 문장
