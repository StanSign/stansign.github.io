변수와 상수

변수 (Variables)

변수의 의미

  • 변수: 데이터를 담을 수 있는 공간
  • 프로그램 동작의 첫 번째 단계는 메모리에 값(데이터)를 저장하는 것
var a = 3 // a에 3을 저장
var b = 7 // b에 7을 저장

var c = b // c에 b의 값을 복사해서 저장

변수의 이름 짓기

  • 소문자로 시작
  • 숫자 사용 가능
    • 하지만 첫글자로 시작하는 것은 금지
  • 특수문자, 한자, 한글도 가능
    • 하지만 관습적으로 사용하지 않음

변수의 선언

  • 변수의 이름 뒤에 :(콜론)을 붙이고 그 뒤에 데이터 타입 지정
var name: String
var a: Int = 3 // 메모리 공간을 생성하는 동시에 데이터 저장

변수의 사용

var name = "홍길동"
var age = 20
var address = "서울 강남구"
print(name, age) // 홍길동, 20

name = "전지현"
print(name, age) // 전지현, 20
print("저의 이름은 \(name)입니다. 저의 나이는 \(age)살 입니다. 그리고 \(address)에 살고 있습니다.") 
// 저의 이름은 전지현입니다. 저의 나이는 20살 입니다. 그리고 서울 강남구에 살고 있습니다.
  • 문자열 보간법(String Interpolation): 문자열 중간에 출력할 변수를 삽입할 때 사용

상수 (Constants)

상수의 의미

  • 상수: 변하지 않는 데이터를 담을 수 있는 공간
let name = "김철수"

name = "우영우" // Error

데이터 타입 (Data Types)

데이터 타입

데이터 타입의 이해

  • 컴퓨터와 사람의 데이터 타입에 대한 이해는 다르다.
    • 주어진 크기와 형태로 공간을 생성한다.
var i: Int = 4
var j: Int = 3
i = 3.4 // Error (Type error)

Swift의 데이터 타입

정수형 데이터 타입
  • Int
    • ex) …, -5, -4, -3, -2, -1, 0, 1, 2, 3, …
실수형 데이터 타입
  • Float
    • 소수점 6자리 / 4-byte
    • ex) -5.324, -4.2211, 0, 1.032, 3.14, …
  • Double
    • 소수점 15자리 / 8-byte
  • Swift에서는 주로 Double 타입 사용
문자형 데이터 타입
  • Char
    • 하나의 문자
    • ex) “A”, “b”, “C”, “한”, “ “(공백)
  • String
    • 문자열
    • ex) “안녕하세요”, “홍길동”, ““(빈 문자열)
참과 거짓
  • Bool
    • true / false

타입 주석 (Type Annotation)

  • 변수를 선언하면서, 데이터 타입도 명확하게 지정하는 방식
var number: Int // 변수 선언
number = 10     // 값을 저장 (초기화)
print(number)   // 10

var a1: Int = 3
var b1: Int = 4

var value: Double = 5.345
value = 12.3
var value2 = value // Double

타입 추론 (Type Inference)

  • 타입을 지정해주지 않아도 컴파일러가 타입을 유추하여 저장하는 방식
var name = "홍길동"
type(of: name) // String:Type

타입 안정성 (Type Safety)

  • 데이터 타입을 명확하게 구분하여 사용하는 속성
    • Swift는 다른 타입의 데이터끼리 계산할 수 없다.
let num5 = 7
let num6: Double = num5 // Error (공간의 크기가 다름)

let d = 12
let e = 3.14
let result = d + 2 // Error  (데이터 타입이 다름)

타입 변환 (Type Conversion)

  • 다른 다입끼리 계산이 안된다면, 타입을 변환하면 된다!
  • 기존의 데이터를 다른 타입으로 변환하여 새롭게 저장
let str = "123"
let number = Int(str)
print(type(of: number)) // Optional<Int>

let str2 = "123.4"
let number2 = Int(str2) // 타입 변환이 실패하면 nil
print(type(of: number2)) // Optional<Int>
print(number2)  // nil

타입 애일리어스 (Type Alias)

  • 기존에 선언되어 있는 타입에 새로운 별칭을 붙여 가독성을 높이는 방법
typealias Name = String
let name: Name = "홍길동"

typealias Something = (Int) -> String
func someFunction(completionHandler: (Int) -> String) {
	
}

func someFunction(completionHandler: Something) {
	
}
