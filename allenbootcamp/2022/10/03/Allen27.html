소멸자

  • 인스턴스가 메모리에서 해제되기 직전 처리되는 내용을 구현하는 메서드
  • 클래스에는 최대 1개의 소멸자만 정의 가능
  • 파라미터 사용하지 않음
  • 소멸자는 모든 속성에 접근할 수 있다.
  • 기본적으로 Swift는 ARC를 통해 자동으로 소멸자를 호출한다.
class Aclass {
	var x = 0
	var y = 0

	deinit {
		print("인스턴스 소멸 시점")
	}
}

var a: Aclass? = Aclass()
a = nil // "인스턴스 소멸 시점"

상속일 때의 문법

  • 상위클래스 소멸자는 하위클래스에 의해 상속
  • 상위클래스 소멸자는 하위클래스 소멸자의 구현이 끝날 때 자동으로 호출
  • 상위클래스 소멸자는 하위클래스가 자체적인 소멸자를 제공하지 않더라도 항상 호출
