Any와 AnyObject를 위한 타입 캐스팅

Any 타입

  • 기본 타입들을 포함하여 커스텀 클래스, 옵셔널, 구조체, 열거형, 함수 타입 등 모든 타입을 포함하여 어떤 타입도 표현할 수 있는 타입
  • 저장된 타입의 메모리 구조를 알 수 없음
    • 타입 캐스팅이 항상 필요
var some: Any = "Swift"
some = 10 // 에러가 나지 않음!
some = 3.2

var someString = some as! String
someString.count

var someString: Any = "Swift"
(someString2 as! String).count
  • 모든 타입을 담을 수 있는 배열을 생성할 수 있다.
let array: [Any] = [5, "Hi", 3.5, Person(), {(name: String) in return name}]
(array[1] as! String).count

AnyObject 타입

  • 클래스 인스턴스만 담을 수 있는 타입
let objArray: [AnyObject] = [Person(), Superman(), NSString()]
(objArray[0] as! Person).name = "최영철"

타입캐스팅에 따른 분기 처리

for (index, item) in array.enumerated() {
	switch item {
	case is Int:
		print("Index \(index): 정수입니다.")
	}
	case let num as Double:
		print("Index \(index): 소수 \(num)입니다.")
	case let person as Person:
		print("Index \(person): 사람입니다.")
		print("이름은 \(person.name)입니다.")
	case is (String) -> String:
		print("Index \(index): 클로저 타입입니다.")
	default:
		print("Index \(index): 그 외의 타입입니다.")
}
